his repository implements a minimal model showing that agency attribution arises solely from inter-hemispheric delay asymmetry.
Core result:


Δτ = τ_R − τ_L = 0 → A(t) ≡ 0 (degeneracy from symmetry)


Δτ > 0 → A(t) > 0 (emergence of signed agency)


Model:
Two symmetric predictors receive identical motor commands; the only structural difference is internal delay (τ_L vs τ_R). The agent compares prediction errors to compute an agency index A(t).
Main analyses:


Δτ sweep: verifies A=0 at Δτ=0 and A>0 for Δτ>0


τ_world robustness: effect holds across generative delays


A(t)–logBF relation: confirms statistical interpretation


2D structure (Δτ × τ_world): identifies sign structure and boundary


Robustness: noise (σ_ξ, σ_e) and learning rate (η) sweeps


Abruptness: transition is discontinuous at Δτ=0

Fixed parameters and explicit sweeps


Multiple random seeds (N_RUNS)

Interpretation:
The vanishing of A at Δτ=0 is not due to linearity but to symmetry under identical update rules and inputs. Delay asymmetry is therefore a necessary structural condition for non-degenerate agency signals.
